## JSX 주의 사항  

1. 중괄호 내부에는 자바스크립트 표현식만 넣을 수 있다
2. 숫자, 문자열, 배열 값만 렌더링 된다
3. 모든 태그는 닫혀 있어야 한다
4. 최상위 태그는 반드시 하나여야 한다  

```jsx
const Main = () => {
    const user = {
        name: "감자",
        isLogin: true,
    };

    if (user.isLogin) {
        return <div>log out</div>;
    } else {
        return <div>log in</div>
    }

    return (
      <>
        {user.isLogin ? (
            <div>로그아웃</div>
        ) : (
            <div>로그인</div>
        )}
      </>
    );
  }

export default Main;
```
